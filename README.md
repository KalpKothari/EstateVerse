# EstateVerse

> A modern AI-powered real estate marketplace connecting buyers, tenants, and property owners through intelligent property discovery, secure verification, real-time communication, and seamless property management.

---

## Overview

EstateVerse is a full-stack PropTech platform built to simplify the entire real estate journey — from discovering properties and scheduling visits to managing listings and communicating with property owners.

The platform combines AI-driven recommendations, role-based workflows, seller verification, real-time messaging, and intelligent search into a single premium real estate ecosystem.

Whether a user is looking to buy their dream home, find a rental, or manage a portfolio of listings, EstateVerse provides a secure and scalable experience powered by modern web technologies.

---

## Core Capabilities

### Intelligent Property Discovery

Finding the right property should feel effortless.

EstateVerse provides:

- AI-powered property recommendations
- Natural language property search
- Location-based discovery
- Lifestyle-focused recommendations
- Budget-aware suggestions
- Commute-optimized listings

Example searches:

```text
Luxury villa in Goa with a pool
3 BHK apartment near metro under ₹1 Cr
Pet-friendly rental in Mumbai
Family home with private garden
```

---

### Real Estate Marketplace

Users can seamlessly:

- Browse properties for sale
- Explore rental listings
- Save properties
- Schedule visits
- Contact property owners
- Track property availability

Supported property types:

- Apartments
- Villas
- Luxury Estates
- Penthouses
- Independent Homes
- Commercial Properties

---

### Seller Verification & Trust System

EstateVerse prioritizes trust and authenticity.

Property owners undergo a structured verification process before being allowed to publish listings.

Verification includes:

- Identity validation
- Ownership verification
- Address verification
- Government-issued documents
- Admin review workflow

This ensures every published listing originates from a verified property owner.

---

### Real-Time Communication

The platform includes a fully integrated messaging experience.

Users can:

- Start conversations directly from property pages
- Communicate with property owners
- Review conversation history
- Receive replies in real time

All conversations persist securely within the platform.

---

### Visit Scheduling & Property Engagement

Users can request property visits directly from listings.

Features include:

- Visit scheduling
- Approval workflows
- Visit tracking
- Status management
- Owner-side visit management

This creates a complete inquiry-to-visit pipeline without leaving the platform.

---

### Property Management Dashboard

Property owners receive a dedicated management workspace.

Capabilities include:

- Property creation
- Listing management
- Availability updates
- Inquiry management
- Visit request handling
- Messaging center
- Verification tracking

Property lifecycle statuses:

- Available
- Under Negotiation
- Sold
- Rented Out
- Temporarily Unavailable
